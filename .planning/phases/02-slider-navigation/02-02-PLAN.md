    - useHashSync: When currentIndex changes, pushes new hash (#project-id) via history.pushState

    - useKeyboardNav: ArrowRight and ArrowDown call nextSlide
    - useKeyboardNav: ArrowLeft and ArrowUp call prevSlide

    - useKeyboardNav: Ignores keyboard events when isAnimating is true
