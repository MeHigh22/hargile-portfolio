  <behavior>
    - deriveTheme({accent, background, gradientFrom, gradientTo}) returns a valid ThemePalette with all 8 fields
    - deriveTheme maps accent -> accent, background -> bg, gradientFrom -> coral, gradientTo -> lavender
    - deriveTheme derives bgElevated as slightly lighter than background (lighten ~5%)
    - deriveTheme derives bgCard as slightly lighter than background (lighten ~8%)
    - deriveTheme derives text as near-white relative to dark background (lighten ~92%)
    - deriveTheme derives textSecondary as a hex color (NOT rgba) for GSAP tween compatibility
    - hexToRgb('#96b8f7') returns [150, 184, 247]
    - lighten('#06080d', 0.05) returns a hex string lighter than input
    - All returned color values are 7-character hex strings (#rrggbb) except textSecondary which may be hex8
  </behavior>
  <action>

    - hexToRgb(hex: string): [number, number, number]
    - rgbToHex(r, g, b): string

    - deriveTheme(core: ProjectColors): ThemePalette — maps 4 core colors to full 8-token palette
